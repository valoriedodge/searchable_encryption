iCTF 2008 Truth File

We are working on identifying a set of ground truth for the traffic captured during the 2008 iCTF competition. This data is ideal for use as a truth file as there were around 40 separate teams that were all tasked with hacking into similar networks. This gives us 40 separate instances against the same target and represents a wealth of attack techniques and strategies.
